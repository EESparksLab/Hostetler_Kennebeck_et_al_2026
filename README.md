#Hostetler_Kennebeck_et_al_2026
All raw data, processed data and R scripts associated with the Hostetler & Kennebeck et al 2026 manuscript are included here.

Data (Folder): All csv files required to run the R script are listed in this folder. 

R Script: All script used to analyze data and generate figures are included in the single R script called Hostetler_et_al_2026.R 

SupplementalData (Folder): All supplementary figures (.pdf) and Tables (.xlsx) associated with the Hostetler & Kennebeck et al 2026 manuscript are available in this folder.

RootPainter (Folder): All composites, which contain the original image and the segmented image are listed here. Example file X1_C1A.jpg indicates Trial 1, Gravity Condition = Control, Plant 1A.
